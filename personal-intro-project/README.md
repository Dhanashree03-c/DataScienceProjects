# Personal Introduction Python Program

## Project Overview
This project is a beginner-friendly Python application that collects a user's personal information such as name, age, city, and hobbies, and displays it in a friendly, formatted output. The objective is to practice Python fundamentals including input/output, variables, lists, and basic logic.

## Project Goals & Objectives
- Learn how to install and run Python programs
- Understand user input using `input()`
- Store and manage data using variables and lists
- Perform basic computations
- Display formatted output using f-strings
- Follow proper GitHub project structure and documentation standards

## Setup Instructions

### Step 1: Install Python
Download and install Python 3.x from:
https://www.python.org/downloads/

Verify installation:
python --version

To run:
python personal_intro.py
